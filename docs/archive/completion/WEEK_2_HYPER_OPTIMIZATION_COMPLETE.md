### 2. Shannon Entropy Analysis

**Formula**:
```
H(X) = -Σ p(xᵢ) log₂ p(xᵢ)

3 Types of Entropy:
  1. Token entropy (which tokens?)
  2. Temporal entropy (when requested?)
  3. Sequence entropy (in what order?)

Combined:
  H_combined = 0.4×H_token + 0.3×H_temporal + 0.3×H_sequence
```

**Example Calculation**:
```
Request history (last 1000):
  Tokens: {BTC: 450, ETH: 350, SOL: 150, MATIC: 50}
  
Token probabilities:
  p(BTC) = 0.45, p(ETH) = 0.35, p(SOL) = 0.15, p(MATIC) = 0.05
  
H_token = -(0.45×log₂0.45 + 0.35×log₂0.35 + 0.15×log₂0.15 + 0.05×log₂0.05)
        = -(0.45×(-1.15) + 0.35×(-1.51) + 0.15×(-2.74) + 0.05×(-4.32))
        = -(-0.52 - 0.53 - 0.41 - 0.22)
        = 1.68 bits

Normalized: 1.68 / log₂(4) = 1.68 / 2 = 0.84
Predictability: 1 - 0.84 = 0.16 (LOW - highly random)

Action: Conservative caching (unpredictable pattern)
```

**Performance**: 2x boost through adaptive scheduling
