## 🎯 How to Enable

### In Railway (5 Minutes)

1. **Navigate:**
   ```
   Railway Dashboard → alchemy-whales → Variables
   ```

2. **Add Variables:**
   ```bash
   # Telegram (easiest)
   TELEGRAM_ENABLED=true
   TELEGRAM_BOT_TOKEN=1234567890:ABC...
   TELEGRAM_CHAT_ID=123456789
   
   # Config
   MIN_ALERT_INTERVAL_SECONDS=30
   MAX_ALERTS_PER_HOUR=50
   FRAUD_RISK_THRESHOLD=60
   HIGH_POTENTIAL_THRESHOLD=80
   ALERT_ON_HIGH_RISK=true
   ALERT_ON_HIGH_POTENTIAL=true
   ALERT_ON_CRITICAL=true
   ```

3. **Save** (auto-deploys)

4. **Verify** in logs:
   ```
   ✅ Alert Notification Service initialized
   ✅ Telegram notifications enabled
   ```

**Done!** Start receiving alerts.
