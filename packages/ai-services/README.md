# Coinet AI Services (ai-services)

> AI/LLM integration, analytics, explainability, and onboarding for Coinet. Smart, extensible, and secure.

---

## ✨ Features
- **AI/LLM Integration**: OpenAI, Anthropic, custom models
- **Analytics**: User analytics, personalized dashboards
- **Explainability**: AI-powered explainability, onboarding
- **Extensible**: Plugin hooks, marketplace, a11y checks

---

## 🚀 Quick Start
1. `cd packages/ai-services`
2. `npm install`

---

## 🏗️ Structure
- `/llm` — LLM integration (OpenAI, Anthropic, etc.)
- `/analytics` — User analytics, personalization
- `/explainability` — Explainability, onboarding
- `/hooks` — Plugin hooks, a11y checks

---

## 📦 Usage Example
```ts
import { suggestTokens } from 'ai-services/llm';
```
