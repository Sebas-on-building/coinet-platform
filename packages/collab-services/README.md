# Coinet Collaboration Services (collab-services)

> Real-time collaboration, presence, onboarding, and extensibility for Coinet. Collaborative, secure, and beautiful.

---

## ✨ Features
- **Real-Time Collaboration**: Yjs, Liveblocks, WebSocket
- **Presence**: User presence, comments, co-editing
- **Onboarding**: Animated, personalized onboarding flows
- **Extensible**: Plugin hooks, analytics, onboarding

---

## 🚀 Quick Start
1. `cd packages/collab-services`
2. `npm install`

---

## 🏗️ Structure
- `/realtime` — Real-time collaboration (Yjs, Liveblocks)
- `/presence` — User presence, comments
- `/onboarding` — Onboarding flows, guides
- `/hooks` — Plugin hooks, analytics

---

## 📦 Usage Example
```ts
import { usePresence } from 'collab-services/presence';
```
