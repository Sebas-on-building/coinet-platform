# Coinet Design Tokens (design-tokens)

> Centralized, versioned, and exported design tokens for Coinet. Live Figma/Sketch sync, atomic, and extensible.

---

## ✨ Features
- **Centralized**: All colors, typography, spacing, motion, etc. in one place
- **Versioned**: Full history, diff, rollback, audit log
- **Exported**: CSS variables, JSON, and more
- **Live Sync**: Figma/Sketch webhook integration
- **Extensible**: Token marketplace, plugin hooks, AI suggestions

---

## 🚀 Quick Start
1. `cd packages/design-tokens`
2. `npm install`

---

## 🏗️ Structure
- `/tokens` — All token groups (color, spacing, etc.)
- `/export` — Exporters for CSS, JSON, etc.
- `/webhooks` — Figma/Sketch webhook handlers
- `/versioning` — Versioning, diff, rollback, audit log

---

## 📦 Usage Example
```ts
import tokens from 'design-tokens/tokens';
```
