# 🚀 ONE COMMAND DEPLOYMENT

## Just Run This:

```bash
./deploy
```

**That's it!** ✅

---

## What It Does:

1. ✅ Builds your code
2. ✅ Commits changes
3. ✅ Pushes to GitHub
4. ✅ Deploys to Railway (if CLI installed)

---

## If Railway CLI Not Installed:

After running `./deploy`, just go to:
**https://railway.app** → New Project → GitHub → Select repo → Done!

---

## Environment Variables:

Set these in Railway dashboard (after first deploy):
- `DATABASE_URL` (required)
- `NODE_ENV=production` (required)

Everything else is optional! ✅

---

**That's it. One command. Done.** 🎉
