# Coinet Mobile App (client-mobile)

> The mobile experience of Coinet. Native, adaptive, atomic, and beautiful—engineered for every device and user.

---

## ✨ Design Philosophy
- **Atomic Design**: Atoms, Molecules, Organisms, Templates, Pages
- **Design Tokens**: Centralized, versioned, live Figma/Sketch sync
- **Device-Adaptive UI/UX**: Inspired by Apple, Canva, TradingView, Solana
- **Accessibility**: WCAG AA+, native a11y, color contrast
- **Themeable & Responsive**: Light/dark, custom themes, foldables, wearables
- **Extensible**: Plugins, widgets, marketplace, AI-driven personalization

---

## 🚀 Quick Start
1. `cd apps/client-mobile`
2. `npm install`
3. `npm run start`
4. Open in Expo Go or your device simulator
