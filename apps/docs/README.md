# Coinet Docs (docs)

> Live documentation, API explorer, design system, and community portal for Coinet. Beautiful, interactive, and always up-to-date.

---

## 📚 Structure
- **Live Docs**: Guides, tutorials, API reference
- **API Explorer**: Interactive, auto-generated from code
- **Design System**: Atomic, themeable, accessible
- **Community Portal**: User dashboards, widgets, themes

---

## 🚀 Quick Start
1. `cd apps/docs`
2. `npm install`
3. `npm run dev`
4. Open [http://localhost:4000](http://localhost:4000)
