# User Experience Context

This bounded context manages personalization, preferences, onboarding, and user interaction for Coinet.

## Features
- Personalization and recommendations
- User preferences and settings
- Onboarding and guided tours
- Interaction pattern analytics
- Event sourcing and projections

## Atomic Modules
- `personalization/` — Recommendation engine
- `preferences/` — User settings, themes
- `onboarding/` — Guided tours, checklists
- `analytics/` — Interaction and engagement metrics
- `events/` — UX events, sourcing, projections

---

All code is atomic, extensible, and tested. See `/docs/` for architecture and onboarding.
