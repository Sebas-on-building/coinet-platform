## Scope of This Log

Records here are limited to **major Phase 2 decisions** that:

- accept a Phase 2 Plan document (Plan 2.0, 2.1, …),
- complete a Phase 2 BTAR (BTAR-003..008+),
- file a Phase 2 transition gate (P2TG-NNN),
- file a Phase 2 readiness review (P2RR-NNN, if used),
- approve a Phase 2 exception (AFE / VSE / SCR),
- complete a Phase 2 FRP or BSCP,
- record an ADR specific to Phase 2.

Routine PR-level work, individual finding state changes, and within-BTAR sub-decisions are **not** logged here; they belong in their respective registries.

## Synchronization Rule

Every entry in this log must also appear in the master decision log at `phase-1/registries/backend-v1-decision-log.registry.md` with the same date and a substantively-equivalent summary. The master log is authoritative across phases; this log is authoritative within Phase 2 detail.
