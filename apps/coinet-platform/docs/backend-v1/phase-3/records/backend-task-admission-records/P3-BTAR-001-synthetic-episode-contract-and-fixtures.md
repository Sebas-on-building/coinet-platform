## 1. One-Sentence Mission

> **P3-BTAR-001 creates the typed synthetic episode contract, expected judgment oracle contract, 3–5 deterministic starter fixtures, and fixture-validation tests required for Phase 3's Backend Judgment Truth Suite.**

### 1.1 Honesty Pin

P3-BTAR-001 is **contract + fixtures + validation only**. It does NOT run any judgment engine, does NOT compare any judgment output, and does NOT touch any production source under `src/api/`, `src/services/judgment/`, `src/services/ai-service.ts`, `src/l13/`, or `src/l14/`. The full corpus (15–25 episodes), truth runner, and semantic assertion engine are out of scope (P3-BTAR-003, -002, -004 territory).

---

## 2. Problem Statement

Plan 3.0 §7–§8 require a synthetic episode + expected judgment oracle data structure before any later Phase 3 task can run. Without P3-BTAR-001, P3-BTAR-002 (runner) and P3-BTAR-003 (corpus) would each invent their own incompatible shapes. P3-BTAR-001 fixes the contract, provides starter fixtures so the structure is concretely exercised, and emits validation helpers so future fixture additions can be checked mechanically.
