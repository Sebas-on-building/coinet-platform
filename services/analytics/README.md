# AnalyticsService

Handles technical indicators, backtesting, and risk analytics.

## Endpoints
- `POST /indicators` - Compute indicators
- `POST /backtest` - Run backtest
- `GET /risk/:userId` - Get risk analytics

## Tech Stack
- Node.js + TypeScript (Express)
- Python (optional, for heavy analytics)
- PostgreSQL/TimescaleDB
