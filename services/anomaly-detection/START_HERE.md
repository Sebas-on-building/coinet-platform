## 💡 Pro Tips

### Tip 1: Start with Baselines
Always learn baselines from historical data before going live:
```typescript
await system.learnHistoricalBaselines(
  DataSource.TRADING_VOLUME,
  historicalData,
  'BTC'
);
```

### Tip 2: Tune Thresholds
Start conservative, tune down:
```typescript
anomalyThresholds: {
  statistical: 3,    // Start at 3σ
  ml: 0.7,          // 70% confidence
  percentile: 95    // 95th percentile
}
```

### Tip 3: Monitor the Monitor
Watch system health and performance:
```typescript
const health = await system.healthCheck();
const stats = system.getStatistics();
console.log(health, stats);
```

### Tip 4: Use Predictions Wisely
Predictions are probabilistic, not certain:
```typescript
if (prediction.confidence > 0.85) {
  // High confidence - prepare action
} else if (prediction.confidence > 0.7) {
  // Medium confidence - monitor closely
} else {
  // Low confidence - informational only
}
```

### Tip 5: Review Autonomous Decisions
Even in auto mode, review periodically:
```typescript
const decisions = tradingAgent.getPendingDecisions();
const performance = tradingAgent.getPerformance();
// Review before approving
```
