# AIService

Handles LLM orchestration, retrieval-augmented generation (RAG), and streaming chat.

## Endpoints
- `POST /analyze` - Analyze chart/project
- `POST /chat` - Streaming chat with AI
- `POST /ingest` - Ingest documents to vector DB

## Tech Stack
- Python + FastAPI (LLM, RAG)
- Node.js + TypeScript (API gateway)
- Pinecone/Weaviate/pgvector (vector DB)
- Google Gemini, Manus, Scispace (LLM APIs)
