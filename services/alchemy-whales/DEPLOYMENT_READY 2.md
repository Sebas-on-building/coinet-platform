## 🚀 Railway Deployment Steps

### Quick Start (5 Minutes)

1. **Open Railway Dashboard:**
   ```
   https://railway.app/dashboard
   ```
   - Select **coinet-platform** project
   - Click **alchemy-whales** service

2. **Add Environment Variables:**
   - Go to: Railway > alchemy-whales > **Variables**
   - Copy variables from: `RAILWAY_ALERT_VARS.txt`
   - Minimum required:
     ```bash
     TELEGRAM_ENABLED=true
     TELEGRAM_BOT_TOKEN=your_bot_token
     TELEGRAM_CHAT_ID=your_chat_id
     ```

3. **Railway Auto-Deploys:**
   - ✅ Detects code push
   - ✅ Builds service
   - ✅ Deploys automatically

4. **Verify Deployment:**
   - Check Railway > alchemy-whales > **Logs**
   - Look for: `✅ Alert Notification Service initialized`
