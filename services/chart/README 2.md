# ChartService

Handles chart configurations, user layouts, and chart image export.

## Endpoints
- `GET /charts/:id` - Get chart config
- `POST /charts` - Create chart config
- `PUT /charts/:id` - Update chart config
- `DELETE /charts/:id` - Delete chart config
- `GET /charts/:id/image` - Export chart as image

## Tech Stack
- Node.js + TypeScript (Express)
- PostgreSQL
- Chart rendering library (e.g., TradingView, Chart.js)
