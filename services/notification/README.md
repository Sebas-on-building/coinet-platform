# NotificationService

Handles alerts, push/email, and in-app notifications.

## Endpoints
- `POST /alerts` - Create alert
- `GET /alerts/:userId` - Get user alerts
- `POST /notify` - Send notification

## Tech Stack
- Node.js + TypeScript (Express)
- Redis (queue)
- Email/SMS/push providers
