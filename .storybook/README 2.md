# Coinet Storybook (.storybook)

> Storybook config, device emulation, design token panel, and live design links for Coinet. Beautiful, interactive, and always up-to-date.

---

## 🚀 Quick Start
1. `npx nx storybook shared-ui` or `yarn storybook`

---

## 🏗️ Structure
- `main.js` — Storybook main config
- `preview.js` — Global parameters, device emulation
- `addons.js` — Addons (a11y, controls, design token panel)

---

## 📦 Usage Example
- View atomic components, molecules, organisms, templates, and pages
- Emulate devices (iPhone, iPad, desktop)
- Live design token panel and Figma/Sketch links
